Common questions

What counts as a scan?

Each unique repository URL submission counts as one scan. Re-scanning the same repo after a fix is a new scan. Pro users have unlimited re-scans.

Can I scan private repositories?

Yes — provide a GitHub access token at scan time. We use it only for that scan and never store it.

How accurate is the score?

Scores are based on 7 weighted dimensions analyzed via static analysis, dependency databases (NVD, OSV), and pattern matching. Calibrated against 200+ real repos. Not a penetration test — treat it as a continuous baseline.

Is my code stored?

We clone the repo temporarily to run the scan, then delete the clone. Findings are encrypted at rest (AES-128-CBC). We never permanently store your source code.

Can I cancel anytime?

Yes. Cancel from your dashboard and your plan stays active until the billing period ends. No cancellation fees.
