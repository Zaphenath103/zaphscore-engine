ZaphScore API

REST API for submitting repository scans, retrieving scores, and exporting reports. Base URL: https://zaphscore.zaphenath.app

Authentication

All endpoints except GET /ping require a Supabase JWT in the Authorization header.

Authorization: Bearer YOUR_SUPABASE_JWT

Get your JWT by signing in via the dashboard or the Supabase JS client. Tokens expire after 1 hour.

Errors

CodeMeaning
401Missing or invalid JWT
404Scan not found
400Invalid request body or scan not complete
429Rate limit exceeded
500Internal error — contact support

Rate Limits

Free: 3 scans total. Pro: 1,000 API requests/day. Enterprise: unlimited. Rate limit headers returned on every response: X-RateLimit-Remaining, X-RateLimit-Reset.

Submit a Scan

POST /api/scans Auth required

Submit a repository URL for scanning. Returns immediately with a scan ID. Poll GET /api/scans/{id} for results.

Request body

FieldTypeDescription
repo_urlrequiredstringGitHub repository URL. Public repos only unless github_token provided.
github_tokenstringGitHub PAT for private repo access. Used only for this scan, never stored.
 
curl -X POST https://zaphscore.zaphenath.app/api/scans \
  -H "Authorization: Bearer $JWT" \
  -H "Content-Type: application/json" \
  -d '{"repo_url": "https://github.com/your-org/your-agent"}'

Response

{
  "id": "scan_abc123",
  "status": "queued",
  "repo_url": "https://github.com/your-org/your-agent",
  "created_at": "2026-04-10T08:00:00Z"
}

List Scans

GET /api/scans Auth required

Returns all scans for the authenticated user, newest first. Supports ?limit=20&offset=0.

Export PDF Report

GET /api/scans/{scan_id}/report.pdf Auth required

Returns a PDF with executive summary, score breakdown table, and full findings list. Scan must be complete. Pro and Enterprise only.

curl -O report.pdf \
  -H "Authorization: Bearer $JWT" \
  https://zaphscore.zaphenath.app/api/scans/scan_abc123/report.pdf

Delete Account

DELETE /api/user/me Auth required

Permanently deletes your account and all associated scans and findings. GDPR Article 17 compliant. Irreversible.

{
  "deleted": true,
  "scans_deleted": 47
}
